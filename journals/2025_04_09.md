		- ((6772c464-77e8-4209-bd31-474acf16c6a8)) 如何判断矛盾是“主要”还是“次要”？
		  collapsed:: true
			- 如果是说标准的话，那么主要矛盾是影响你能不能及格、能不能生存、会不会违约的矛盾，而次要矛盾是影响你能不能优秀、能不能舒适、能不能拿奖金的矛盾。
			- 主要矛盾解决能不能活的问题，次要矛盾解决活不活得好的问题。
			- 所以总要先抓主要矛盾。而且如果这两个目标有所矛盾，总要先无条件的偏向前者。
			- 这看上去非常直白，似乎是废话，但显然很多人看上去用不好这个法则，这主要是因为两条：
			- 1）因为经验浅薄、理论混乱而搞错了哪些问题能致命、哪些问题不重要。不是不知道要抓主要矛盾，问题是主要矛盾找错了。
			- 2）你区分的主次矛盾和老师 / 父母 / 配偶 / 朋友 / 路人的不一样，互相说服不了。
			- 所以你想要一个完美的理论工具，能帮你确凿无疑的、有充分说服力的证实主要矛盾是什么，解决自己的问题、也解决自己与这些人的争论。
			- 先说这个理论工具——这个工具是有的，简单来说就是第一性原理和大历史。
			- 第一性原理说起来玄妙，但在本质上就是物理主义，也就是“一切在最终意义上都是由物理定律决定的的”。一件事可不可行、概率如何、效率如何，归根结底是物理定律一锤定音。化学规律、生物规律、心理规律、社会规律……归根到底是物理规律的综合表达。可以说人分两种，一种是遇事能瞬间从目标到底层物理规律建立完整链路的人，可以列出微观粒子的运动如何最终影响到消费者决策、影响到大选结果的每一步，建立起预测模型。
			- 这说穿了是一个有很多变量，而输出值为“是 / 否”的函数。一旦列出了这个函数，就可以基于这个数学关系，评估各个参数对事情的最终影响，并基于影响这些参数各自所需要成本，确定一个最优的分配资源的方案。
			- 最后这个方案所呈现的结果，总是“主要的预算要分配到某几个关键要素上，次要预算分配到另一些值得照顾的、影响较小的次关键要素上，将其他成本过度高昂的、无法全面顾及的、期望比较乐观的要素托管到老天爷手里，靠机动预算随机应变”。
			- 这几乎是必然的结局。
			- 而在这个必然结局的基础上，就出现了大历史的参与。
			- 大历史是通过对大量史实的掌握遂形成的一种历史观，一种对时代潮流向何处去、为何向此处去的总体观点。简单来说，就是历史看多了，会发现历史是有巨大惯性的。因此就像一切其它有巨大惯性的对象一样，历史的短期和中期趋势是可以预测的，而历史的远期趋势也是可以形成有说服力的观点的。
			- 在这里，大历史掌管着那些“成本过度高昂的、无法全面顾及的、期望比较乐观的要素”。
			- 你可能会觉得这些要素最多不过占三成，只要你物理数学功底强大，甚至可以把这三成都纳入管理，投入预算控制起来。但只要你是这么想，就意味着你的物理数学还学的稀巴烂，还在胡思乱想。
			- 因为你只要学明白了就会发现，在事关经营、事业的大问题上，这类要素起码要占五成以上。因为如果这类要素低于一成，这个问题多半就会被自动化、由可以严格执行程序的软硬件去完成了；如果不到五成，这个问题就会被交给有标准流程和赔偿条款的外包劳动力去完成；只有这类要素超过五成的事情，才真正的只能由主角自己扛在肩膀上，自己去赌运气。
			- 也就是说——一件事情之所以会让你纠结什么是主要矛盾，首先就在于其中有五成以上的因素，你必须基于对历史的认识将其挑出来、留出机动预算并托付给命运。
			- 是在这个基础上，你才能对剩下的那五成不到的要素分主次，拿肉眼盯。
			- 所以尽管理论上你只需要充分掌握物理定律和数学规律，但在实践上反而是历史观先切走一大半蛋糕，然后才轮到数学技能和物理知识来切剩下的蛋糕——这就是为什么人必须关心“宏大叙事”并尊重和追求经验，以及人为什么严重的依赖信仰。
			- 这样也解释了为什么作为年轻人这个困扰很大——就是因为你在学校里做题，几乎是涉及不到“历史观切掉的那一大块不确定性”部分的。都是先通过理想化、规范化处理好的干净的问题，通过简单的规律运用就能解决的问题。而一旦走向社会，要独自面对人生选择，由历史观、“宏大叙事”所托管的那一大块不确定性就不再能逃避。
			- 而你在这方面毫无训练和准备。
			- 你的客观体验就是你基于学校训练做的计划，总是频频的因为“宏大叙事”和“陈旧经验”被上司、领导甚至客户否决。
			- 最经典的现象，就是你划分的重点要素，被上司告知“并不重要，不用去管”，因为在ta的划分里，这个要素被划到了托管要素里。
			- 而你甚至连“交由历史托管”的概念都没有，对你来说没有任何要素是可以“交由历史托管”的，因为你就觉得这就是赌运气、赌命。这从逻辑上就注定了一定会有你认为极为重要的要素不断的被上司告知“不用管”。
			- 在不明就里的前提下，你会本能的将这种挫折视为尊严的损失，并进而将“迷信经验”“沉迷于宏大叙事”“墨守成规”“规训”这类的罪名往胆敢冒犯你的尊严、破坏你的自信的敌人们身上扣去，把本来应该补上的历史缺课、应该寻找到的自己的“宏大叙事”（这实际上往往是历史观的一种误称）的时间和精力，用在了“反抗”和“寻找反抗同志”上。
			- 甚至可以这么说——恰恰有很多学校教育并不出色的学生因为没有这种“知见障”，比较“听话”，而从实践中更快的补全了对历史托管部分的视觉。而又因为历史托管部分往往实际上是整个问题最首要、最大的一块，导致ta们反而比只会用理论计算处理规划化后的问题的学校精英发展得更好、社会地位更高——也就是所谓的“体脑倒挂”现象。
			- 其实哪有什么“体脑倒挂”？那只是因为人类的理性认知还远没有到达单纯凭理论推算就能替代经验判断的水平——甚至可能永远也到达不了这个水平。
			- 因为我们的理性认知水平越高，本身就会导致我们观察历史的视力变强，从而导致历史篇幅本身迅速膨胀并进而导致历史经验在决策中的地位也随之水涨船高，其决定性地位再次得到巩固。
			- 很多年轻人幻想着科学越发达，经验越次要，越不用理会老师傅们的经验直觉，却发现科学越发达，青年教师和“学阀”之间的地位差距几乎没有变化——因为“学阀”们一样也变得更强大了，ta们的经验也建立在更发达的基础之上了。
			- 毕竟人家多活几十年不是白活的。
			- 最后总结——分清主次矛盾的基本方法在于应用物理规律完成数学建模，但要高效、优美的完成这个建模，历史观水平至关重要，否则你的模型本身就会扭曲和残缺。
			- 综合下来——钻研历史、尊重前辈，等待属于你自己的时候到来。
		- ((6772c464-77e8-4209-bd31-474acf16c6a8)) 行为远远配不上理想，我该怎么办？
		  collapsed:: true
			- 你有亏负什么和人签订的契约、约定的承诺吗？有违反什么法律、合同、社会责任之类的社会契约吗？
			- 没有的话，就不要平白无故的对自己搞什么“鄙视”、“羞耻”之类的迫害。
			- 注意，你很可能低估了前两者的分量——实际上你要是真的认真看待前两者，你真的没有什么亏欠的概率几乎不存在。你之所以觉得自己谁都不欠、谁都对得起，基本肯定是因为你吧“对得起”的标准放得比较低——或者用你的话来说，比较“合理”——的结果。
			- 所以，一个健康的人其实只看前两条就已经有足够的惭愧和忏悔要承担了。
			- 已经不需要在后一个领域再挖掘什么羞耻感来“自我激励”了。
			- 严格意义上讲，人只有对不起别人才需要感到羞耻、惭愧。只有对不起别人，才有所谓的过失、罪孽。
			- 你有权“对不起”自己，不会也不必对自己觉得亏欠和罪孽。
			- 那些觉得自己对自己亏欠、罪孽的人，其实说的是一个自己想象中的虚拟的可能性，是自己创造的一个幻想的自我。
			- 他们在想象是自己这样那样的行为，使得这个想象的自我无法出生、无法享受“本来应该有”的快乐，于是从这个思考上对自己产生愤恨。
			- 问题是，这种“映像设定”的权柄即使存在，也只能是传说中全知全能、主宰万物和一切历史的神的权力，而不是身为一介凡人的你所能妄想拥有的权力。
			- 你在设想“本可以存在的那个更好的我”的时候已经犯了自居上帝的基本错误，后续的所有负面反应都是这个错误带来的自然法的惩罚。
			- 而最坏的结果是啥呢？不是你受了惩罚觉得痛苦不堪，而是这种痛苦没能抵消你窃取神权而获得的隐秘的快乐，没能打断你一犯再犯的毒瘾。
			- 这是一个自然法陷阱，你懂不懂？
			- “设想可能存在的自我”本身就带着强烈的快感，这快感天然的被设计成可以压倒紧随其后的罪孽感、羞愧感，于是人类会一犯再犯。
			- 一犯再犯，一犯再犯，一犯再犯……
			- 直到这些漫不经心被沉淀的罪孽感、羞愧感，凝固成低贱感、绝望感，直到给这些偷摸权杖的小屁孩来个十一抽杀，让别的小孩知道知道厉害。
			- 到时候一抓一个准——你看你摸得满手都是青黑，你还想赖？还不排排靠墙站，老老实实抽生死签？
			- 抽一回，没抽到你，旁边那个同批的倒霉蛋咣叽挨了大巴掌，命都丢了半条，你就知道怕了。
			- 你还摸不摸？
			- 自己尽力，做到啥样是啥样，学会“无论结果如何，永远既为这个结果由衷满意和自豪、又对更高的境界产生更强烈的向往”，不沾一丁点基于完美自我妄想的自怨自艾，这就是你的全部责任。
			- 你不需要再判自己一个什么“本来可以更好”的罪，你敢判，你就等于报名参加明年的十一抽杀大奖赛。
			- 学会穿过一些蠢蛋的嘲讽侮辱，为一个四条腿都不平整的手工小板凳高兴得手舞足蹈，为个脸都是歪的素描作业切实感到“我进步好多”的欢喜，而又不为了苟得这个欢喜去做“我有我的审美”、“都是别人不识货”的自欺欺人，这是一个极难极难的功夫。
			- 你不参加几次十一抽杀、不挨几个狠的，都很难理解这种不偏不倚的正念是何等的宝物。
			- 所以你挨了几轮也不要哭天抢地，这就是自然法教育人的风格。这是很操蛋，问题是自然法就这种可恶的恶趣味——你拿ta没办法。
			- 一轮轮的努力，一轮轮的证实自己的正念笃定，这本身就会在具体的成绩之外给你极大的信心，以至于具体的得失根本无足轻重。
			- 我脚下就是天道，这条道上只有一时难以领会的胜利，没有失败，我的未来必定光明。
			- 我对我自己，没有对不起。
